AI enabled Soil Analysis

Solution: Developed AI based mobile application with bluetooth integration with Sensor device to provide real time soil health and diagnostic solution. The application connects with bluetooth with instruments for real time reading of 9 critical parameters from the soil. Based on these values, the application provides AI based analysis and diagnostic details to the farmers to improve their yield.

Key Features

  • Integration of AI models and building the self learning tool.
  • Integration with the sensor device using Bluetooth for real time data handling.
  • Multi language capabilities to support regional languages.
  • Supporting different stakeholders requirements using a role based approach.

Key Benefits

  • Realtime health report of the soil using the mobile app. Eliminating the need for sending the sample to the lab and waiting for 1-2 weeks for the results.
  • AI enabled diagnostic reports, helping farmers to optimize use of the critical resources like water, fertilizer and pesticide in real time.
Technology Stack: Google Cloud Platform (GCP), Firebase database, Flutter.
